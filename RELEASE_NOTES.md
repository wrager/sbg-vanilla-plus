# v0.2.2

## Багфиксы

- Переписан модуль «Вращение карты одним пальцем»:
  - Точки теперь подгружаются после поворота карты (расширение extent до диагонали вьюпорта)
  - Исправлено срабатывание жеста: touch events на viewport вместо pointer events на document
  - Блокировка перетаскивания карты во время жеста вращения
  - Центр вращения учитывает смещение карты (совместимость с модулем «Смещение центра карты»)
