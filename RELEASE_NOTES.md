# v0.2.0

## Новые фичи

- **Новый модуль: Вращение карты одним пальцем** — в режиме СЛ/FW можно вращать карту одним пальцем, не ломая штатный двухпальцевый поворот
- **Перехват ошибок** — логирование ошибок консоли и необработанных исключений с сохранением в диагностику
- **Кнопка «Сообщить об ошибке»** — в настройках, открывает GitHub Issue с автоматически заполненной диагностикой
- **Чекбокс «Переключить все»** — массовое включение/выключение модулей с тремя состояниями

## Улучшения UI

- Локализован заголовок окна настроек
- Переработан индикатор необходимости перезагрузки модуля
- Кнопка закрытия настроек перенесена вниз
- Ограничена максимальная ширина списка настроек
- Улучшен модуль «Улучшенный UI попапа точки»

## Багфиксы

- **Модуль «Сдвиг центра карты вниз»** — заменён CSS-хак на штатный OL `view.padding`, исправлен рендеринг точек в padding-зоне
- **Модуль «Отключить зум по двойному тапу»** — больше не ломает двухпальцевый поворот карты

## Прочее

- Шаблоны GitHub Issues (баг-репорт и запрос фичи)
- Обработка sync/async ошибок модулей с различением фаз init/enable/disable
- Отображение ошибок модулей на экране настроек
