# v0.7.1

## Исправления

- Исправлено залипание попапа точки при включённом модуле «Закрытие попапа свайпом»: после серии свайпов точки могли переставать открываться, экран затемнялся, а попап не появлялся.
- Свайп-закрытие попапа работает стабильнее при повторных открытиях и переключении между точками.
