# v0.9.0

## Новое

- **Избранные точки** — звезда в попапе точки добавляет её в избранное, список шарится с CUI, ключи избранных не удаляются автоочисткой. В настройках модуля — экспорт и импорт списка в JSON для переноса между устройствами.
- **Удаление ключей автоочисткой** — раньше было отключено; теперь доступны режимы «Быстро» (при изучении точки) и «Медленно» (вручную через кнопку во вкладке ключей в ОРПЦ).

## Улучшения

- **Улучшенный главный экран** — компактная верхняя панель: ник под кнопками, инвентарь в ОРПЦ, шестерёнка вместо «Настройки». Кнопка настроек SVP перенесена в игровое меню.
- **Нгрс-зум** — чувствительность в 1.5 раза выше, зум плавный без ступенек. Отдельный модуль «Отключить double-tap zoom» убран — его функцию выполняет Нгрс-зум.
- **Keep Screen On** — в SBG Scout удержанием экрана управляет сам хост, модуль в настройках показывается строкой «Реализовано в SBG Scout».
- **Прочее** — падение одного модуля не ломает остальные и панель настроек; toast при отказе localStorage; терминология OPS → ОРПЦ в русских описаниях.
