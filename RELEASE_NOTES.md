# v0.2.3

## Улучшения

- Модуль «Сдвиг центра карты вниз» больше не требует перезагрузки при включении/выключении — работает мгновенно через переключение флага

## Багфиксы

- Исправлен статус модулей с асинхронным `enable()`: статус `ready` теперь выставляется только после завершения инициализации, а не сразу
- Модуль «Экран не гаснет»: корректно показывает `status: failed` при недоступном Wake Lock API (ранее оставался `ready`)
- Исправлено падение модуля «Вращение карты одним пальцем» при отсутствии padding у view
