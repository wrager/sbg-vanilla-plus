# v0.5.0

## Новые модули

- **Группировка тостов ошибок** (`groupErrorToasts`) — одинаковые тосты с ошибками объединяются в один со счётчиком повторений вместо спама множеством отдельных уведомлений
- **Автоочистка инвентаря** (`inventoryCleanup`) — автоматическое удаление лишних ключей из инвентаря по настраиваемым правилам
- **Ключи на карте** (`refsOnMap`) — просмотр и удаление ключей прямо на карте

## Изменения

- Модуль «Сворачиваемая панель» переименован в **«Улучшенный главный экран»** (`enhancedMainScreen`) — добавлено центрирование кнопки атаки
- Добавлен скрипт скачивания референсов (`npm run refs:fetch`)

## Исправления

- Исправлено отображение цвета переполнения инвентаря в свёрнутой панели
- Исправлено ложное открытие попапа точки при вращении карты одним пальцем и при зуме двойным тапом
- Исправлен зум по двойному тапу на мелких масштабах карты
