# v0.10.0

## Новое

- **Инструменты рисования** — новый модуль от [pr0head](https://github.com/pr0head): рисование линий и треугольников по точкам, редактирование и точечное удаление, выбор цвета, привязка к ближайшим точкам, импорт/экспорт схем для обмена между игроками. Кнопка модуля размещается на карте под кнопкой регионов, тулбар адаптируется под узкие экраны.

## Улучшения

- **Совместимость с SBG 0.6.1** — скрипт работает на 0.6.0 и 0.6.1. На 0.6.1 автоматически отключаются модули, функции которых игра реализовала сама: «Избранные точки», «Автоочистка инвентаря», «Количество ключей на точках», «Зарядка при полном заряде», «Нгрс-зум», «Вращение карты одним пальцем», «Переход к следующей точке» — собраны в отдельную группу в настройках с указанием причины. Свайп для закрытия попапа отключается, чтобы не конфликтовать с нативным жестом игры между точками. На неподдерживаемой версии (новее 0.6.1) скрипт спрашивает подтверждение перед запуском.
- **Тосты** — клик по тосту закрывает его, не дожидаясь таймера. Клик по тосту больше не проходит сквозь нижележащий UI и не вызывает действий под ним.
