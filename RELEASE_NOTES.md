# v0.1.0

Первый релиз SBG Vanilla+!

Модульный userscript для улучшения UI/UX игры SBG. Каждый модуль можно включить или отключить в панели настроек (кнопка ⚙ внизу экрана).

## Стилизация

- **Сворачиваемая верхняя панель** — сворачивает верхнюю панель, показывая только инвентарь и кнопку ОРПЦ/OPS
- **Улучшенный UI попапа точки** — увеличенные кнопки на экране точки для удобства на мобильных
- **Сдвиг центра карты вниз** — сдвигает центр карты вниз, чтобы видеть больше карты впереди по ходу движения
- **Количество ключей на точках** — показывает число ключей для каждой видимой точки на карте
- **Увеличенная область нажатия** — увеличивает кликабельную область точек на карте

## Фичи

- **Отключить зум по двойному тапу** — предотвращает случайный зум при быстрых нажатиях
- **Экран не гаснет** — не даёт экрану гаснуть во время игры (Wake Lock API)

## Багфиксы

- **Фикс кнопки рисования** — кнопка Draw всегда активна, исправляет баг игры с зависанием в неактивном состоянии
