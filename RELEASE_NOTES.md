# v0.11.0

## Совместимость

- Скрипт переведён на SBG 0.6.1. На других версиях покажет подтверждение перед запуском.

## Новое

- **Миграция избранного** — переносит локальный список избранных точек SVP/CUI в нативные «звёздочки» или «замочки» SBG 0.6.1. Импорт/экспорт legacy-списка в JSON сохранены. Точки без ключей в инвентаре пометить нельзя — нативный флаг живёт на стопке ключей, не на самой точке.
- **Анимация свайпа к следующей точке** — попап улетает в направлении свайпа когда есть следующая точка, отскакивает обратно когда нет.
- **Закрытие меню действий ключа** — меню «Favorite»/«Lock»/«Manage» в карточке ключа закрывается после клика по любой из кнопок.
- **Фикс залипания кнопки починки** — возвращает кнопку «Починить» из зависшего неактивного состояния.
- **Фикс кнопок после переключения** — восстанавливает срабатывание первого тапа по кнопкам попапа точки после переключения на следующую.
- **Фикс текста количества ключей** — счётчик ключей в подписи точки на карте обновляется сразу после изучения точки, автоочистки и удаления через «Ключи на карте».

## Улучшения

- **Переход к следующей точке** — теперь работает свайпом влево/вправо в попапе вместо кнопки. Приоритет по полезности (свободные слоты, доступная для изучения, ближайшая) и трекинг посещений сохранены.
- **Свайп для закрытия попапа** — закрывает попап свайпом вверх (раньше было влево/вправо, конфликтовало с нативным горизонтальным свайпом игры).
- **Автоочистка инвентаря** — защищает ключи точек, помеченных нативным «замочком» SBG. Пока активна, принудительно отключает нативный «Сборщик мусора», чтобы две системы не удаляли ключи параллельно. До завершения миграции избранного автоудаление ключей блокируется. Появилась кнопка «Очистить ключи» во вкладке ключей инвентаря.
- **Ключи на карте** — защищены ключи точек с нативным «замочком». Кнопка «На карте» перенесена к нативной кнопке сортировки.
- **Вращение карты одним пальцем** — подавляется во время нативного жеста зума двумя пальцами (двойной тап и перетаскивание).
- **Фикс кнопки рисования** — Draw-карусель и счётчик ключей на кнопке обновляются при переключении на следующую точку.

## Реализовано нативно в игре

- **Зарядка при полном заряде** — кнопка «Починить» в SBG 0.6.1 теперь сама учитывает заряд.
- **Нгрс-зум** — двойной тап и перетаскивание для зума добавлены в нативную карту.
- **Количество ключей на точках** — игра показывает счётчик через канал «References» в Layers > Text.
- **Избранные точки** — заменены на нативные «звёздочки» и «замочки». Перенос старого списка — через модуль «Миграция избранного».
