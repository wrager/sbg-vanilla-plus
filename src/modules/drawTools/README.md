# Инструменты рисования (drawTools)

Модуль добавляет поверх карты инструменты для рисования и редактирования схем (линии из 2 точек и треугольники из 3 точек) с привязкой вершин к точкам и импортом/экспортом между игроками.

## Что получает игрок

- Кнопка модуля в виде floating OL-control под `.region-picker` (рядом с зум-кнопками карты), иконка пера
- Floating toolbar с SVG-кнопками:
  - линия (ровно 2 точки, автозавершение после второй)
  - треугольник (ровно 3 точки, автозавершение после третьей)
  - редактирование (перо)
  - удаление одной фигуры по клику (мишень с крестом)
  - выбор цвета (нативный color-picker)
  - привязка всех вершин к ближайшим видимым точкам (волшебная палочка)
  - копирование схемы в буфер (две рамки)
  - импорт схемы из текста (стрелка вверх)
  - полная очистка с confirm (корзина)
  - закрытие toolbar (крест)
- Отмена незавершённого рисования — клавиша `Escape` или повторное нажатие на инструмент рисования
- Схемы сохраняются в localStorage и отображаются при загрузке страницы

## Формат данных

- Канонический формат хранения: **IITC DrawTools JSON**
- localStorage key: `svp_drawTools`
- Поддерживаемые типы в MVP: `polyline`, `polygon`
- `marker` и `circle` в MVP не поддерживаются

## Архитектура

- Карта: `getOlMap()` из `src/core/olMap.ts`
- Слой рисования: отдельный `Vector` layer c `name='svp-draw-tools'`
- Сериализация: `iitcFormat.ts` (parse/stringify + type guard)
- UI: floating OL-control под `.region-picker` + toolbar в `document.body`. Позиция control'а пересчитывается через `getBoundingClientRect()` от picker'а; MutationObserver и ResizeObserver следят за DOM/resize, чтобы control оставался прилипшим
- Иконки кнопок: свои inline-SVG из `drawToolsIcons.ts`, наследуют `currentColor` от темы
- Cleanup в `disable()` снимает слой, интеракции, toolbar, OL-control, observer'ы и стили

## Ограничения MVP

- Нет маркеров и кругов
- Нет выбора формата импорта/экспорта (только IITC JSON)
- Snap привязывает каждую вершину к ближайшей видимой точке на карте; если точек нет — вершина остаётся на месте
