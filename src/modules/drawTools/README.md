# Инструменты рисования (drawTools)

Модуль добавляет поверх карты инструменты для рисования и редактирования схем линий/полигонов.

## Что получает игрок

- Кнопка `DT` в верхнем левом меню (сразу после `settings`)
- Floating toolbar с режимами:
  - `L` — линия (ровно 2 точки, автозавершение после второй)
  - `P` — треугольник (ровно 3 точки, автозавершение после третьей)
  - `E` — редактирование
  - `D` — удаление по клику
  - `S` — привязка всех вершин к ближайшим видимым точкам
  - `C` / `V` — копировать / вставить JSON схемы
  - `R` — полная очистка (с confirm)
- Отмена незавершённого рисования — клавиша `Escape`

## Формат данных

- Канонический формат хранения: **IITC DrawTools JSON**
- localStorage key: `svp_drawTools`
- Поддерживаемые типы в MVP: `polyline`, `polygon`
- `marker` и `circle` в MVP не поддерживаются

## Архитектура

- Карта: `getOlMap()` из `src/core/olMap.ts`
- Слой рисования: отдельный `Vector` layer c `name='svp-draw-tools'`
- Сериализация: `iitcFormat.ts` (parse/stringify + type guard)
- UI: кнопка в `.game-menu` + toolbar в `document.body`
- Cleanup в `disable()` снимает слой, интеракции, toolbar, кнопку и стили

## Ограничения MVP

- Нет маркеров и кругов
- Нет выбора формата импорта/экспорта
- Нет KML
- Snap привязывает каждую вершину к ближайшей видимой точке на карте; если точек нет — вершина остаётся на месте
