# Улучшенный UI вкладки ключей (enhancedRefsTab)

Делает карточку ключа во вкладке References инвентаря визуально предсказуемой: содержимое разнесено на три фиксированные строки независимо от длины имени владельца. Высота карточки одинаковая для всех ключей — список не «прыгает» по вертикали.

## Реализация

Только CSS (`styles.css`). Flexbox-разметка `.inventory__item-left` с `flex-direction: column` и фиксированной высотой строк. Никакого JS — модуль injectStyles на enable, removeStyles на disable.

## Файловая структура

| Файл                      | Назначение                                          |
| ------------------------- | --------------------------------------------------- |
| `enhancedRefsTab.ts`      | Определение модуля: enable/disable, инжект/съём CSS |
| `enhancedRefsTab.test.ts` | Тесты (метаданные, styles inject/remove)            |
| `styles.css`              | 3-line flex-разметка карточки                       |

## Настройки

Модуль не имеет настроек. Включён по умолчанию.
