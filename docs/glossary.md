# Словарь терминов

Термины и сокращения, используемые в проекте и в игре SBG.

## Игровые термины

| Термин                                      | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| ------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| СЛ / FW                                     | Режим следования за игроком на карте (localStorage `follow`)                                                                                                                                                                                                                                                                                                                                                                                                 |
| ОРПЦ / OPS                                  | Кнопка инвентаря в верхней панели (`#ops`)                                                                                                                                                                                                                                                                                                                                                                                                                   |
| Точка / Point                               | Игровая локация на карте                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| Схема / Draw plan                           | Набор нарисованных пользователем фигур (линий и треугольников) поверх карты                                                                                                                                                                                                                                                                                                                                                                                  |
| Вершина / Vertex                            | Угловая точка нарисованной линии или треугольника                                                                                                                                                                                                                                                                                                                                                                                                            |
| Ключ / Ref                                  | Предмет инвентаря, привязанный к точке (`type: 3` в inventory-cache)                                                                                                                                                                                                                                                                                                                                                                                         |
| Ядро / Core                                 | Предмет для простановки на точке (`type: 1` в inventory-cache)                                                                                                                                                                                                                                                                                                                                                                                               |
| Катализатор / Cat                           | Предмет для атаки точки (`type: 2` в inventory-cache)                                                                                                                                                                                                                                                                                                                                                                                                        |
| Метла / Broom                               | Спецпредмет для мгновенного уничтожения точек в радиусе действия игрока (`type: 4`)                                                                                                                                                                                                                                                                                                                                                                          |
| Простановка / Deploy                        | Установка ядер на точке                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| Изучение / Discover                         | Хак точки для получения лута (ядер, катализаторов, ключей)                                                                                                                                                                                                                                                                                                                                                                                                   |
| Зарядка / Repair                            | Восполнение энергии ядер на точке (кнопка «Починить» / «Recharge»)                                                                                                                                                                                                                                                                                                                                                                                           |
| Нгрс / Ngrs                                 | Игра, которой вдохновлена SBG                                                                                                                                                                                                                                                                                                                                                                                                                                |
| Избранная точка / Favorited point           | Локальная пометка точки в SVP/CUI (IndexedDB `CUI/favorites`). После SBG 0.6.1 заменена нативной звёздочкой/замочком на ключе; локальный список служит источником миграции через модуль `favoritesMigration`                                                                                                                                                                                                                                                 |
| Звёздочка / Favorite (нативная)             | Нативный флаг в SBG 0.6.1 (бит 0 поля `f` стопки в `inventory-cache`): визуальное маркирование ключа в инвентаре; не защищает от удаления автоочисткой                                                                                                                                                                                                                                                                                                       |
| Замок / Lock (нативный)                     | Нативный флаг в SBG 0.6.1 (бит 1 поля `f` стопки в `inventory-cache`): защищает ключ от попадания в линии рисования и от удаления через модули `inventoryCleanup` (автоочистка + slow-кнопка) и `refsOnMap` (viewer на карте). Lock-флаг агрегируется per-point: одна стопка locked → вся точка защищена                                                                                                                                                     |
| Сборщик мусора (нативный)                   | Нативный механизм SBG 0.6.1: чекбокс `usegrb` плюс лимиты по уровню в настройках игры. Сервер раз в N минут чистит инвентарь по этим лимитам. Дублирует функцию модуля `inventoryCleanup`, поэтому пока `inventoryCleanup` активен, нативный сборщик принудительно отключается через `nativeGarbageGuard`: серверный POST `/api/settings { usegrb: false }` плюс DOM-disable инпутов с подписью «Заменён модулем автоочистки Vanilla+» в нативных настройках |
| Ограничения рисования / drawingRestrictions | Модуль, скрывающий цели из списка рисования: избранные (целиком или только последний ключ), дальние, и все кроме центра звезды                                                                                                                                                                                                                                                                                                                               |

## Сокращения проекта

| Сокращение | Расшифровка                                                                 |
| ---------- | --------------------------------------------------------------------------- |
| СБГ / SBG  | Название игры (Скромная Браузерная Геолокационка / Simplest Browser Geoloc) |
| SVP        | SBG Vanilla Plus — этот проект                                              |
| OL         | OpenLayers — картографическая библиотека, используемая игрой                |
| CUI        | SBG Custom UI — референсный скрипт сообщества                               |
| EUI        | SBG Enhanced UI — референсный скрипт сообщества                             |
| DT         | DrawTools — инструмент рисования схем поверх карты                          |

## Правила использования

- В пользовательских текстах (release notes, README, описания модулей) использовать русские варианты: **СЛ**, **ОРПЦ**, **точка**, **ключ**
- Допустимы двуязычные варианты: **СЛ/FW**, **ОРПЦ/OPS**
- В коде (имена переменных, комментарии) использовать английские варианты
- Термин **точка** относится к игровой локации, **вершина** — к углу нарисованной фигуры. Не путать.
