# Словарь терминов

Термины и сокращения, используемые в проекте и в игре SBG.

## Игровые термины

| Термин        | Описание                                                             |
| ------------- | -------------------------------------------------------------------- |
| СЛ / FW       | Режим следования за игроком на карте (localStorage `follow`)         |
| ОРПЦ / OPS    | Кнопка операций в верхней панели (`#ops`)                            |
| Точка / Point | Игровая локация на карте                                             |
| Ключ / Ref    | Предмет инвентаря, привязанный к точке (`type: 3` в inventory-cache) |
| Простановка   | Игровое действие на точке (deploy)                                   |

## Сокращения проекта

| Сокращение | Расшифровка                                                  |
| ---------- | ------------------------------------------------------------ |
| SBG        | Название игры (Sector Battle Game)                           |
| SVP        | SBG Vanilla Plus — этот проект                               |
| OL         | OpenLayers — картографическая библиотека, используемая игрой |
| CUI        | SBG Custom UI — референсный скрипт сообщества                |
| EUI        | SBG Enhanced UI — референсный скрипт сообщества              |

## Правила использования

- В пользовательских текстах (release notes, README, описания модулей) использовать русские варианты: **СЛ**, **ОРПЦ**, **точка**, **ключ**
- Допустимы двуязычные варианты: **СЛ/FW**, **ОРПЦ/OPS**
- В коде (имена переменных, комментарии) допустимы английские варианты
